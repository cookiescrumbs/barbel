## Parameters

| Key   |      Values      | Description |
|----------|-------------|-------------|
| items | `[{name: 'Apple', url: '/apple'}, {name: 'pear', url: '/pear'}]` | Items to search through and navigate to. |

## Usage

```js
import SimpleCard from 'simple-card';

const props = {
};

<SimpleCard {...props}/>
```
